# Outputs

## Overview

This folder stores outputs generated during model training, evaluation, and experimentation for the **OmniReview** project.

The outputs help monitor model performance, compare different model configurations, and support reproducibility of experimental results.

---

## Planned Contents

### Model Checkpoints

Saved model weights for:

- Variational Autoencoder (VAE)
- Transformer (T5 + LoRA)
- Diffusion Model
- GAN Discriminator
- Normalizing Flow

---

### Generated Reviews

Sample review headlines and review bodies generated by different model configurations.

---

### Training Logs

Training and validation logs, including:

- Training loss
- Validation loss
- Learning curves
- Training history

---

### Evaluation Results

Performance summaries for all experiments, including:

- Conditional Perplexity
- BERTScore
- Headline–Body Similarity
- Rating–Sentiment Agreement
- Helpfulness Calibration Error
- Embedding-space FID
- GAN Discriminator AUC

---

### Visualizations

Figures generated during experimentation, including:

- t-SNE/UMAP latent space visualizations
- Learning curves
- Performance comparison plots
- Evaluation metric charts

---

## Repository Notes

Large datasets, model checkpoints, and generated outputs may not be stored in this GitHub repository due to storage limitations.

Representative samples, visualizations, and documentation will be included whenever appropriate to ensure reproducibility.
