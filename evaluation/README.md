# Evaluation

This folder contains scripts and documentation used to evaluate the performance of the OmniReview framework.

## Planned Evaluation Metrics

- Conditional Perplexity
- BERTScore
- Headline–Body Similarity
- Rating–Sentiment Agreement
- Helpfulness Calibration Error
- Embedding-space FID
- GAN Discriminator AUC

Evaluation scripts and results will be added as model development progresses.
