# Transformer (T5 + LoRA)

## Purpose

This module implements the Transformer component of the OmniReview framework using the pretrained T5-small model with Low-Rank Adaptation (LoRA).

## Responsibilities

- Generate review headlines
- Generate review bodies
- Condition generation on latent representations learned by the VAE
