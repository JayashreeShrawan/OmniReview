# Normalizing Flow

## Purpose

The Normalizing Flow module models the probability distribution of review helpfulness conditioned on review attributes.

It predicts helpfulness scores using exact likelihood estimation.

## Why Normalizing Flows?

Normalizing Flows provide invertible transformations that enable efficient density estimation and probabilistic modeling.

## Planned Implementation

The Flow module will include:

- RealNVP architecture
- Coupling Layers
- Density Estimation
- Helpfulness Prediction
